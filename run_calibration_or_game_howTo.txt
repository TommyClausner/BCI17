The following describes how to use the shell scripts run_calibration.sh and run_game.sh

The scripts will run a "headless" version of MATLAB in the terminal and execute the script of interest. After a 10s delay the same applies to the python part.

### run calibration ###

#use: sh run_game.sh <matlabroot path> <path to script to execute> <python interpreter path> <python script path>

sh run_calibration.sh /Applications/MATLAB_R2015a.app /Users/Tommy/Github_repositories/BCI17/feedback_sig.m /Users/Tommy/anaconda/bin/ipython /Users/Tommy/Github_repositories/BCI17/calib_stim.py

### run game ###

sh run_game.sh /Applications/MATLAB_R2015a.app /Users/Tommy/Github_repositories/BCI17/feedback_sig.m /Users/Tommy/anaconda/bin/ipython /Users/Tommy/Github_repositories/BCI17/brainflyTest.py
